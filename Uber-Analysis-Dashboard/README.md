# Uber Analysis Dashboard

## Overview

This Power BI dashboard analyzes Uber booking data and provides insights into booking performance, revenue, customer behavior, vehicle performance, ratings, and location trends. The dashboard is designed to help stakeholders monitor key business metrics and make data-driven decisions.

## Tools Used

- Power BI
- Excel

## Business Requirements

The dashboard includes the following analyses:

### Overview
- Completed Bookings
- Lost Bookings
- Revenue
- Total Distance
- Average Distance
- Vehicle Filter
- Monthly Booking & Revenue Analysis
- Quarterly Booking & Revenue Analysis
- Revenue by Vehicle Type
- Top Pickup & Drop Locations
- Average Rider & Driver Ratings

### Vehicle Analysis
- Booking Count by Vehicle
- Revenue by Vehicle
- Vehicle Contribution

### Revenue Analysis
- Revenue by Customer
- Revenue by Vehicle
- Revenue by Payment Method
- Monthly & Quarterly Revenue Trends

### Rider Analysis
- Ride Cancellation Reasons
- Payment Method Analysis
- Monthly & Quarterly Analysis
- Rider Details
- First-Time, Returning & Regular Riders

### Location Analysis
- Monthly Total Distance
- Distance by Vehicle
- Busy Time Slots
- Busy Areas
